# Papers

Use this folder for paper metadata, reviewed summaries, and extraction notes
that may later become typed wiki cards.

Do not assume every paper can be redistributed. Even when a PDF is publicly
available, check the license, publisher policy, and author rights before
committing the file itself. Prefer storing:

- title, authors, venue, year
- DOI or URL
- access date
- license or redistribution note
- short paraphrased summary
- reviewed claims with page/figure/table references
- extraction artifacts that are allowed to be shared

Full paper PDFs should normally stay local and untracked unless redistribution
rights are explicit.
