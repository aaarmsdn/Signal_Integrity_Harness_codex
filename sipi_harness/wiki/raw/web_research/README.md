# Web Research

Store curated web research notes here.

Prefer compact JSON, YAML, or Markdown records with:

- source title
- URL
- access date
- publisher/organization
- source tier candidate
- short paraphrased summary
- useful design claims
- related topics
- whether the source is allowed for design rules, background, or context only

Do not copy long article text. Use links, short excerpts when legally allowed,
and your own summary.
